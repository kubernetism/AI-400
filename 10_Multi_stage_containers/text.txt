 Key Design Decisions

  - Backend: Uses uv sync --frozen in builder, copies only .venv to runtime. Runs as non-root appuser. Entrypoint:
  uvicorn main:app
  - Frontend: 3-stage build (deps -> build -> runtime). Copies only .next/standalone + static assets. Runs as
  non-root nextjs user
  - Compose: Backend has a health check; frontend waits for it via depends_on: condition: service_healthy. Both on a
  shared app-network

  To Build & Run

  # Add yourself to docker group (one-time)
  sudo usermod -aG docker $USER
  newgrp docker

  # Build and start
  docker compose build
  docker compose up

  Frontend at http://localhost:3000, backend at http://localhost:8000.
