# Event Planning Reference

Use when the plan is for organizing an event: conference, workshop, wedding, Eid gathering, hackathon, etc.

---

## Event Planning Phases

### 6+ Weeks Before
- [ ] Define event goal, audience, and format
- [ ] Set date, time, venue
- [ ] Estimate attendance and budget
- [ ] Send save-the-dates / invitations
- [ ] Book vendors (catering, AV, decoration)
- [ ] Create run-of-show document

### 2–4 Weeks Before
- [ ] Confirm RSVPs
- [ ] Finalize program/agenda
- [ ] Brief all staff/volunteers
- [ ] Prepare materials (badges, handouts, signage)
- [ ] Confirm logistics (transport, parking, accessibility)
- [ ] Set up registration system

### 1 Week Before
- [ ] Final vendor confirmations
- [ ] Rehearsal / dry-run
- [ ] Prepare contingency plans (weather, tech failure)
- [ ] Day-of communication plan (WhatsApp group, etc.)

### Day Before
- [ ] Venue setup
- [ ] Tech check (projector, mic, WiFi)
- [ ] Volunteer briefing

### Day Of
- [ ] Arrive 60–90 min early
- [ ] Registration desk setup
- [ ] Execute run-of-show
- [ ] Designate point person for issues

### Post-Event
- [ ] Thank-you messages to speakers/attendees
- [ ] Collect feedback (Google Form, WhatsApp poll)
- [ ] Financial reconciliation
- [ ] Document lessons learned

| Category | Typical % of Budget |
|---|---|
| Venue | 25–35% |
| Catering/Food | 25–35% |
| AV & Technology | 10–15% |
| Decoration | 5–10% |
| Marketing/Invitations | 5–10% |
| Contingency Buffer | 10–15% |

## Islamic Event Considerations

For Eid gatherings, PIAIC events, or community functions:
- Schedule around Fajr + Isha (don't overlap prayer times)
- For Eid: plan around Eid prayer timing (typically 7:30–9:00 AM)
- Provide prayer space / musalla if event is > 3 hours
- Halal food confirmation from all vendors
- Gender considerations for seating/program if required
- Jumu'ah: avoid Friday 12:00–2:00 PM slot
